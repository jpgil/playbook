## Objetivo de este repositorio
Promover la idea de Playbooks personales que puedan descargarse y usarse como guías para proyectos nuevos. En un principio, contenidos técnicos: como usar docsify y github, trucos de mermaid, etc. Pero puede irse expandiendo a otros temas como escritura literaria, gestión de empresas, etc.
