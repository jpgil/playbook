# AGENTS.md

Repositorio de playbooks personales de JPGIL. Gestionado con MaC (Management as Code).

## Lectura obligatoria

Al inicio de cada sesión o conversación, leer `MaC/procesos/amigo-MaC.md` **antes de cualquier otra acción**. Ese archivo define tus modos de operación y te indica qué procesos consultar según el tipo de solicitud.

## Tipos de trabajo

1. **Gestión** — Gobernada por `MaC/ritual_de_sesion.md` (se accede vía `procesos/amigo-MaC.md`). Sesiones, log, roadmap, planificación.
2. **Operativo** — Guiado por el `README.md` del repositorio. Antes de implementar un ítem del Plan de Acción, consultar el índice de guías del playbook por si hay una relevante.
3. **Skills del agente** — Capacidades propias (herramientas, búsqueda, ejecución). Usar libremente dentro del contexto de 1 y 2.

## Límites

### Preguntar primero
- Hacer git push.
- Renombrar o mover archivos.
- Modificar `MaC/ritual_de_sesion.md`, `AGENTS.md` o `.cursorrules`.

### Nunca hacer
- Modificar el `LICENSE`.
- Eliminar archivos sin confirmación explícita.
- Commitear credenciales o información sensible.
